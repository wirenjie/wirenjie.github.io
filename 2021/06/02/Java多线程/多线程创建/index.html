¶实现Runnable接口（重）

步骤：

  1. 定义MyRunnable类实现Runnable接口
  2. 实现run()方法，编写线程执行内容
  3. 创建线程对象，调用start()方法启动线程

注意：

  1. 推荐使用Runnable对象，因为Java单继承的局限性
  2. 和继承Thread类的区别就是开启线程的方法上有区别

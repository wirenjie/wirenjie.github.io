Linux基础命令

¶目录处理命令

¶ls

  • 原意list，所在路径/bin/ls

  • 执行权限：所有用户

  • 功能：显示目录文件

  • 语法：ls 【选项】 【目录或文件】

  • 选项参数

    • -a，显示所有文件，包括隐藏文件
    • -l，详细显示信息
    • -d，查看目录属性（不显示文件信息）
    • -h，文件大小显示增加可读性
    • -i，查看文件i节点

¶mkdir

  • 原意make directories，路径/bin/mkdir

  • 执行权限：所有用户

  • 功能：创建新目录

  • 语法：mkdir 【选项】 【目录名1】【目录名2】

  • 常用参数

    • -p，递归创建

¶cd

  • 原意 change directory，shell内置命令

  • 执行权限：所有用户

  • 功能：切换目录

  • 语法：cd 【目录】

  • 常用参数

    • …回到上一级目录
    • -，回到上个所在目录

¶pwd

  • 原意print working directory，所在路径/bin/pwd
  • 执行权限：所有用户
  • 功能：显示当前目录路径
  • 语法：pwd

¶rmdir

  • 原意remove empty directories，路径/bin/rmdir
  • 执行权限：所有用户
  • 功能：删除空目录
  • 语法：rmdir 【目录名】

¶cp

  • 原意copy，路径/bin/cp

  • 执行权限：所有用户

  • 功能：复制文件或目录

  • 语法：cp 【选项】 【原文件/目录】【目标目录】

  • 常用参数

    • -p，保留文件属性复制
    • -r，复制目录

  • 注意：可以同时对多个文件或目录进行复制

¶mv

  • 原意move，路径/bin/mv
  • 执行权限：所有用户
  • 功能：剪切文件、改名
  • 语法：mv【选项】 【原文件/目录】【目标目录】
  • 注意：可以同时对多个文件或目录进行剪切

¶rm

  • 原意rm，路径/bin/rm

  • 执行权限：所有用户

  • 功能：删除文件或目录

  • 语法：rm【选项】 【原文件/目录】

  • 常用参数

    • -r，删除目录
    • -f，强制执行

¶文件处理命令

¶touch

  • 原意touch，路径/bin/touch
  • 执行权限：所有用户
  • 功能：创建空文件
  • 语法：touch 【文件名】

¶cat

  • 原意cat，路径/bin/cat

  • 执行权限：所有用户

  • 功能：显示文件内容

  • 语法：cat 【文件名】

  • 常用参数

    • -n，显示行号

  • 注意tac为反向显示（参考学习）

¶more

  • 原意more，路径/bin/more

  • 执行权限：所有用户

  • 功能：分页显示文件内容

  • 语法：more 【文件名】

  • 操作

    • 空格或f，翻后页
    • B或b，往前翻页
    • 回车，换行
    • q或Q，退出

¶less

  • 原意less，路径/usr/bin/less

  • 执行权限：所有用户

  • 功能：分页显示文件内容（可搜索）

  • 语法：less 【文件名】

  • 操作

    • 空格或page down，翻后页
    • page up，往前翻页
    • 回车，换行
    • q或Q，退出
    • /XXX，可搜索文件中相关内容，n向下查找，N反向查找

¶head

  • 原意head，路径/usr/bin/head

  • 执行权限：所有用户

  • 功能：显示文件前面十行内容

  • 语法：head 【文件名】

  • 操作

    • -n，显示指定行数

¶tail

  • 原意tail，路径/usr/bin/tail

  • 执行权限：所有用户

  • 功能：显示文件后面十行内容

  • 语法：tail 【文件名】

  • 操作

    • -n，显示指定行数
    • -f，动态显示文件末尾内容

¶权限管理命令

¶chmod

  • 原意change the permissions mode of a file

  • 路径/bin/chmod

  • 执行权限：所有用户

  • 功能：改变文件或目录权限

  • 语法

    • chmod + 【{ugoa} {±=} {rwx}】【文件或目录】

      u：代表所有者
      g：代表所属组
      o：代表其他人
      a：代表所有人

    • chmod 【mode = 421】【文件或目录】

  • 常用参数

    • -R，递归修改

¶其他权限管理命令

  • chown

    • 原意change file ownership

    • 路径/bin/chown

    • 执行权限：root用户

    • 功能：改变文件或目录的所有者（用户存在）

    • 语法

      • chown 【用户】 【文件或目录】

    • 常用参数

      • -R，递归修改

  • chgrp

    • 原意change file group ownership

    • 路径/bin/chgrp

    • 执行权限：root用户

    • 功能：改变文件或目录的所属组

    • 语法

      • chgrp 【用户组】 【文件或目录】

  • umask

    • 原意the user file-creation mask

    • shell内置命令

    • 执行权限：所有用户

    • 功能：显示、设置文件的缺省权限

    • 语法

      • umask 【-S】 【数值】
      • 以rwx形式显示新建文件的缺省权限

    • 注意

      • 针对目录默认权限为755，文件权限为644
      • umask结果为0022，umask -S结果为755
      • 022是需要和777做与运算的，也就是755

¶权限划分说明

  • r可读–4，w可写–2，x可执行–1

  • rwxrw-r–：对应二进制和764

  • 对文件来说

    • r：可以查看文件内容
    • w：可以修改文件内容
    • x：可以执行文件

  • 对目录来说

    • r：可以列出目录中的内容
    • w：可以在目录创建、删除文件
    • x：可以进入目录

¶链接命令

¶ln

  • 原意link，路径/bin/ln

  • 执行权限：所有用户

  • 功能：生成链接文件

  • 语法：ln 【选项】 【原文件】 【目标文件】

  • 常用参数

    • -s，创建软连接

  • 链接概念

    • 软链接

      • 类似于快捷方式，指向性符号链接
      • 源文件丢失后无法运行
      • i节点和源文件不同

    • 硬链接

      • 拷贝且与源文件同时更新
      • 源文件丢失后可以访问
      • i节点与源文件相同
      • 不能跨分区建立，不能针对目录建立

¶用户管理命令

¶useradd

  • 所在路径/usr/sbin/useradd
  • 执行权限：root用户
  • 功能：添加新用户
  • 语法：useradd 【用户名】

¶passwd

  • 所在路径/usr/bin/passwd

  • 执行权限：所有用户

  • 功能：设置用户明码

  • 语法：passwd【用户名】

  • 注意

    • root用户可以无视密码复杂度修改，普通用户不行
    • 普通用户只能更改自己的密码，root用户可修改任何用户密码

¶who

  • 所在路径：usr/bin/who
  • 执行权限：所有用户
  • 功能：查看登录用户信息
  • 语法：who
  • tty为本地终端登录，pts为远程终端登陆
  • 注：用w命令能得到更详细的信息

¶帮助命令

¶man

  • 原意manual，所在路径/usr/bin/man

  • 执行权限：所有用户

  • 功能：获取帮助信息

  • 语法：man 【命令或配置文件】

  • man ls，查看ls命令的帮助信息

  • man services，查看配置文件信息（不用绝对路径）

  • 当命令和配置文件重名时会优先查看命令

    • 1，代表命令
    • 5，代表文件
    • 查看文件使用 man 5 XXX

¶其他命令

  • whatis，只查看命令做什么用的信息
  • apropos，查看配置文件用途信息（不用绝对路径）
  • 命令 --help，查看命令的常见选项信息
  • help，查看shell内置命令信息

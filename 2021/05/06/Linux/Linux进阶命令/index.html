Linux进阶命令

¶关机重启命令

¶shutdown

  • 语法：shutdown 【选项】 时间

  • 如：shutdown -h 【（now）/（20：30）】

  • 常用参数

    • -c，取消设置的定时关机
    • -h，关机
    • -r，重启

¶其他关机命令

  • halt
  • poweroff
  • init 0

¶其他重启命令

  • reboot
  • init 6

¶补充：系统运行级别，init命令调用

  • 0，关机
  • 1，单用户（启用最小服务）
  • 2，不完全多用户，不含NFS服务
  • 3，完全多用户
  • 4，未分配
  • 5，图形界面
  • 6，重启
  • cat /etc/inittab可以查看
  • runlevel命令也可查看当前及上一次运行级别

¶logout

  • 退出登录

¶网络命令

¶write

  • 所在路径/usr/bin/write
  • 执行权限：所有用户
  • 功能：给用户发信息，以Ctrl+D保存结束
  • 语法：write 【用户名】
  • 注：只能是在线用户间发送

¶wall

  • 原意write all，所在路径/usr/bin/wall
  • 执行权限：所有用户
  • 功能：给所有用户发广播信息
  • 语法：wall 【信息】
  • 注：所有在线用户都能收到，包括本机用户

¶ping

  • 所在路径/bin/ping
  • 执行权限：所有用户
  • 功能：测试网络连通性
  • 语法：ping 【IP地址】
  • -c，指定发送次数

¶ifconfig

  • 原意interface configure
  • 所在路径/sbin/ifconfig
  • 执行权限：root用户
  • 功能：查看和设置网卡信息
  • 语法：ifconfig 【网卡名称】 【IP地址】
  • 注：此命令是临时设置的网络地址

¶mail

  • 所在路径/bin/mail

  • 执行权限：所有用户

  • 功能：查看发送电子邮件

  • 语法：mail 【用户名】

  • Ctrl+D键结束内容发送

  • 操作

    • mail界面下，help可以查看帮助信息
    • 输入邮件序号可以查看邮件
    • h键查看列表，d+序号删除，q键退出

¶last

  • 所在路径/usr/bin/last
  • 执行权限：所有用户
  • 功能：列出过去登录系统的用户信息
  • 语法：last
  • 注：lastlog命令可以看所有用户最后登录时间

¶traceroute

  • 所在路径/bin/traceroute
  • 执行权限：所有用户
  • 功能：显示数据包到主机间的路径
  • 语法：traceroute 【IP地址】

¶netstat

  • 所在路径/bin/netstat

  • 执行权限：所有用户

  • 功能：显示网络相关信息

  • 语法：netstat 【选项】

  • 常用参数

    • -t，TCP协议
    • -u，UDP协议
    • -l，监听
    • -r，路由
    • -n，显示IP地址和端口
    • -a，查看所有

  • 常见用法

    • netstat -tlun，查看本机监听端口
    • netstat -an，查看本机所有网络连接
    • netstat -rn，查看本机路由表

¶setup

  • 所在路径/usr/bin/setup

  • 执行权限：root用户

  • 功能：配置网络

  • 语法：setup

  • 注意

    • 只有redhat系统存在的命令
    • 配置完后需要service network restart
    • 此命令永久生效

¶mount

  • 所在路径/bin/mount

  • 执行权限：所有用户

  • 功能：挂载命令

  • 语法：mount 【-t 文件系统】【设备文件名】【挂载点】

  • mount -t iso9660 /dev/sr0(或/dev/cdrom) /mnt/cdrom

  • 注意：

    • umount /dev/sr0，取消挂载
    • -t iso9660可以省略
    • sr0和cdrom是一个文件，都是默认好的，不需要修改
    • 取消挂载不能在挂载目录下进行

¶压缩解压命令

¶gzip

  • 所在路径/bin/gzip
  • 执行权限：所有用户
  • 功能：压缩文件，压缩后格式为.gz
  • 语法：gzip 【文件】

¶gunzip

  • 所在路径/bin/gunzip
  • 执行权限：所有用户
  • 功能：解压缩格式为.gz的文件
  • 语法：gunzip 【文件】
  • gzip -d 【文件】也可以解压缩
  • 注：Linux只能压缩文件，压缩后源文件会消失

¶tar

  • 所在路径/bin/tar

  • 执行权限：所有用户

  • 功能：打包目录，打包后格式为.tar.gz

  • 语法：tar 【选项】【压缩后文件名】【目录】

  • 常用参数

    • -c，打包
    • -v，显示详细信息
    • -f，指定文件名
    • -z，打包同时压缩/解压缩
    • -x，解包
    • 注：-f一定要写在最后，不然-f后的参数会被作为文件名

¶zip

  • 所在路径/usr/bin/zip
  • 执行权限：所有用户
  • 功能：压缩文件或目录，格式为.zip
  • 语法：zip 【选项】【压缩后文件名】【文件/目录】
  • -r，压缩目录

¶unzip

  • 所在路径/usr/bin/unzip
  • 执行权限：所有用户
  • 功能：解压缩格式为.zip的压缩文件
  • 语法：unzipzip 【压缩文件】

¶bzip2

  • 所在路径/usr/bin/bzip2
  • 执行权限：所有用户
  • 功能：压缩文件,格式为.bz2
  • 语法：bzip2 【选项】【文件】
  • -k，压缩文件后保留原文件
  • 注：tar -cjf 压缩文件为.tar.gz2

¶bunzip2

  • 所在路径/usr/bin/bunzip2
  • 执行权限：所有用户
  • 功能：解压缩.bz2的压缩文件
  • 语法：bunzip2 【选项】【压缩文件】
  • -k，解压缩后保留原文件
  • 注：tar -xjf 解压.tar.bz2文件

¶文件搜索命令

¶find

  • 原意find，所在路径/bin/find

  • 执行权限：所有用户

  • 功能：查找和搜索文件

  • 语法：find 【搜索范围】 【匹配条件】

  • 常用参数

    • find /etc -name init，目录/etc中查找文件init

      • 不会搜索出包含init的文件，只会显示为init名的文件
      • 需要模糊搜索需要用*通配符或？单个字符匹配
      • 不区分大小写需要使用-iname

    • find / -size +204800，根目录下查找大于100MB的文件

      • 减号表示小于多大的文件
      • 数字的单位是数据块，一个是512字节

    • find /home -user test，根目录下查找所有者为test的文件

    • find /etc -cmin -5，/etc下查找5分钟内被修改过属性的文件和目录

      • -amin，访问时间access
      • -cmin，文件属性change
      • -mmin，文件内容modify
      • 加号表示超过多长时间

    • -a，两个条件同时满足

      • find /etc -size +163840 -a -size -204800
      • /etc下查找大于80MB小于100MB的文件
      • -o，两个条件满足一个即可

    • -exec {} \ ;，对搜索结果执行操作

      • find /etc -name test -exec ls -l {} ;
      • 在/etc下查找test文件并显示其详细信息
      • -ok {} \ ;，对执行的操作需要逐一确认是否执行
      • {}表示对查找结果进行整合，\为转义符，;为结束

    • -type，根据文件类型查找

      • f，文件
      • d，目录
      • l，软链接文件

    • -inum，根据i节点进行查找

¶其他搜索命令

  • locate

    • 原意locate，所在路径/usr/bin/locate

    • 执行权限：所有用户

    • 功能：在文件资料库中查找文件

    • 语法：locate 【文件名】

    • 注意

      • 更新时间较长，无法查找实时新文件
      • 除非root用户下updatedb更新资料库
      • /tmp这种临时目录下的文件不在资料库收录范围

    • -i，不区分大小写

  • which

    • 原意which，所在路径/usr/bin/which
    • 执行权限：所有用户
    • 功能：搜索命令所在目录及别名信息
    • 语法：which 【命令】

  • whereis

    • 原意whereis，所在路径/usr/bin/whereis
    • 执行权限：所有用户
    • 功能：搜索命令所在目录及帮助文档路径
    • 语法：whiereis 【命令】

  • grep

    • 原意grep，所在路径/usr/bin/grep

    • 执行权限：所有用户

    • 功能：在文件中搜素字符匹配的行并输出

    • 语法：grep -iv 【指定字符】【文件】

      • 如：grep mysql /root/install.log

    • 参数

      • -i，不区分大小写

      • -v，排除指定字串

        • 如果只排除行首有此字符的行，需要在字符前加^
